Common HTML Tags

These are the most commonly used HTML tags and a short description of each.

Block Elements

Block elements take up their own line and take up the full width of the page.

a

This tag defines a hyperlink.

Body

Defines the HTML's body.

Div

Makes a section in the HTML prominent.

Main

Specifies the content of the HTML.

Footer

Defines the footer for a section.

Li

Defines an item on a list.

Head

Defines info from the HTML.

Section

Defines a part in the HTML.

Article

Defines an artiel in an HTML.

p

Defines a paragraph in an HTML.

html

The root of an HTML document.

Header

Defines the header for a document or section.

Nav

Defines navigation links in an HTML.

Aside

Defines content aside from the HTML content.
